## TRABAJO REALIZADO

El presente trabajo busca resolver dos tareas fundamentales de la Gestión Escolar:
- Seguimiento y Actualización del Plantel Docente y No Docente de las Escuelas.
- Gestión y Procesamiento de los sueldos del Personal.

En el ARCHIVO Proyecto_Final.pdf se encuentra detallado el Proceso Realizado y el contenido Resumido de la Base de Datos

La entrega está organizada en Carpetas, para poder organizar el trabajo realizado. 
Por un lado se encuentra la carpeta `Inserts` que contiene los Script de inserción y los archivos `.CSV` que se utilizaron para la carga.
Por otro lado se presenta la carpeta `Objetos`, la cual contiene los scripts de creación de Triggers, Procedimientos, Vistas y Funciones.
